Toon een doorzoekbare prompt-bibliotheek op elke WordPress-pagina via een shortcode.

== Description ==

De Prompt Library plugin laat je een overzicht van AI-prompts tonen op je WordPress-site.
Gebruik de shortcode `[prompt_library]` op elke pagina of post.

== Installation ==

1. Upload de map `prompt-library` naar `/wp-content/plugins/`.
2. Activeer de plugin via 'Plugins' in WordPress.
3. Gebruik `[prompt_library]` in een pagina of post.
4. Ga naar **Prompt Library** in het admin-menu voor instellingen.

== Shortcodes ==

`[prompt_library]` — Toont de volledige prompt-bibliotheek.
